# ibm_db2_script_automation
IBM DB2 - Python Script to execute IBM DB2 Queries

** This Project is made with Gemini AI **

# What has been done?
1. Created a Docker IBM DB2 Container
2. Used Python Script to Connect to that DB and execute Query

# IMPORTANT
1. For ease of use, use this Project in Linux
2. As of time of writing Python 3.11 is the best Version for the IBM DB2 v11.5 with 'ibm_db' 3.2.3 Python Package
3. Latest version of Python does not have fully compatible packages (use older versions)
4. The CLI Driver that 'ibm_db' python package uses should be higher or equal to that of IBM DB2 Version installed
